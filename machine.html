Machine Learning 101



Por: Ing. Edwin Jiménez

Actual desarrollador en Relativity6, egresado del Instituto Tecnológico de Tepic, Edwin Jiménez abrió su conferencia con la definición de Learning y el termino IA.
Comentó que la IA actualmente se encuentra en cualquier página web de servicios, como por ejemplo Netflix, Google, Amazon y asistentes como Siri o bien autos como Tesla.
Mencionó que algunas la IA se divide en diversas áreas las cuales son:
Machine Learning.
NLP.
Experts Systems.
Vision.
Speech.
Planning.
Robotics.

Y a su vez Machine Learning se subdivide en:
Deep Learning.
Supervised.
Unsupervised.

Machine Learning tiene 3 fases las cuales son entrada, proceso, salida y los algoritmos mas populares supervisados son perceptron y red neuronal.

Algunas aplicaciones de Machine Learning son las siguientes:
Automatic Machine Translations.
Artistic Filters.
Representación artística de imágenes (GoogleMet).
Clasificación de Objetos.
Generación automática de texto.
Generación automática de escritura a mano.

Algunas herramientas para redes neuronales son: TensorFlow, Seikits learn, Spark Mlib, Lasagne.




Juan Daniel Medrano Barajas - ITTepic 2017
