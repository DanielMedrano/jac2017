Tendencias en el Desarrollo Web



Por: Ing. Adam Carrasco

Adam Carrasco es actual desarrollador de software en Oracle, comenzó su conferencia hablando de que era web, posteriormente se enfoco al Front End y el Back End, dentro del Front End comento que se encontraba el código HTML, CSS y JS, mientras que dentro del Back End se tiene DBs y WebServices.

Menciono las Progressive Web Apps así como también las Single Page Applications. Posteriormente se enfocó a los editores y entornos de desarrollo que actualmente se usan, dentro de los cuales estaban Atom, SublimeText y Visual Studio.

Por ultimo terminó su conferencia con una charla y aconsejando 10 tips que se deben de saber antes de graduarse, los cuales son los siguientes:

  1. Aprender a amar un lenguaje de programación.
  2. Saber estructuras de datos.
  3. Entrenar las habilidades de programación de algoritmos.
  4. Utilizar frameworks y saber como estos trabajan internamente.
  5. Conocer la diferencia de git, github y su interaccion.
  6. No parar solo por ser un proyecto escolar.
  7. No esperarse hasta 9no semestre para adquirir experiencia.
  8. No solamente quejarse.
  9. Intentar crear tu propio negocio.
  10. Leer, Leer y aprender Ingles.
